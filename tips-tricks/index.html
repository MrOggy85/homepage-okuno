Tips & Tricks

  • Syncthing

    Sync your files firectly between your devices. No need for Dropbox, Google Drive, iCloud etc.

  • bat

    cat clone with syntax highlighting and Git integration

  • The Fuck

    corrects your previous console command.

  • xz

    node lib that wraps bash command. Making it easier to write bash scripts using JavaScript.

  • File Browser

    File Browser is a create-your-own-cloud-kind of software where you can install it on a server, direct it to a path and then access your files through a nice web interface.

  • Filestash

    Manage your data online. Filestash lets you access and manage your data in a simple way regardless of where that data is stored
